# Part 02

Topics Covered:

- Variables:
These are the names given to a memory location in a program.

- Data Types:
They specify the type of data. Classified as integers, floating point numbers, strings, boolean and none type.

- Operators:
Classified as arithmetic, assignment, comparison, and logical operators.

- Type Casting:
type() is used to identify the type of the data, where as typecasting helps us convert from one data type to another.

- User Input:
Helps us take input from the user. By default, the input is always taken as a string.

- Basic practice exercises:
1. Finding sum of 2 numbers
2. Modulo operator program
3. Type of the input given by the user
4. Comparison operator example program
5. Calculating average of two numbers
6. Calculating square of a number
