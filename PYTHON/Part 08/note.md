Topics Covered:

- Function:
A group of statements performing a specific task. When the code becomes complex, it is easier when we divide the code into blocks. Using functions increases the reusability and readability of the code. A function is divided into function call, function definition, and function body.

- Types of functions:
1. Built in functions
2. User defined functions

- Arguments:
These are the values you pass to the function. We have default parameter which gets executed when no parameter is passed.

 - Recursion:
A function calling itself is known as recursion. This basically divides the program into smaller parts where the same function is called.
