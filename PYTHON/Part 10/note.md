# Part 10

Topics Covered:

- Object oriented programming:
It is one among of the popular approaches of solving a problem by creating objects. This approach focuses on using reusable code.

- Class:
Blueprint for creating an object

- Object:
Instantiation of a class. When a class is defined, a template is defined. Memory is allocated only after object instantiation.

 - Class and instance attributes:
An attribute that belong to the class is a class attribute, and an attribute that belongs to an object is an instance attribute.

- Self parameter:
It refers to the instance of the class.

- Static method:
A function that does not use a self parameter. (@staticmethod)

- __init__() constructor:
It is a special methid which first runs as soon as the object is created. It is also known as constructor. It takes the self parameter and can take further parameters too.

- Basic practice exercises:
1. Create a class Programmer for storing information of few programmers working at Microsoft
2. Write a class Calculator capable of finding square, cube and square root of a number
3. Create a class with a class attribute a; create an object from it and set ‘a’ directly using object.a = 0. Does this change the class attribute?
4. Add a static method in problem 2, to greet the user with hello
5. Write a Class Train which has methods to book a ticket, get status (no of seats) and get fare information of train running under Indian Railways
6. Can you change the self-parameter inside a class to something else?
