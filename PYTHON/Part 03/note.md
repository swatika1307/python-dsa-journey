# Part 03

Topics covered:

- Strings:
Strings are sequence of characters enclosed in quotes. They may be single quotes, double quotes, or triple quotes.

- Slicing:
Index goes from 0 to length-1. Slicing syntax [start:end:skip]. Important is the concept of negative slicing - always try to convert the negative indices into positive indices by adding the length.

- String functions:
There are a lot of string functions such as len, endswith, startswith, count, find, lower, upper, capitalize, title, find, replace, strip, lstrip, rstrip etc.

- Escape sequences:
They deal with characters like \n, \t, \', \", \\ etc.

3. Finding double spaces in the string
4. Replacing double spaces with single spaces in the string
5. Playing around with escape sequences
