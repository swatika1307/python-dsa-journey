# Part 11

Topics Covered:

- Inheritance:
It is a way of creating a new class from an existing class.

- Types:
1. Single
2. Multiple
3. Multi-level

- super():
It is used to access the methods of a super class in the derived class.

 - Class method:
It is a method which is bound to the class and not to the object of the class. Marked with @classmethod.

- Property decorators:
Used to make a method behave like an attribute.

- Getter:
A method decorated with @property is called a getter. It is used to access an attribute while allowing additional logic to be applied if required.

- Setter:
Used to modify an attribute while allowing validation or additional logic before updating the value.

- Operator overloading:
Allows us to define how operators behave for objects of a user defined class.

- Basic practice exercises:
1. Create a class (2-D vector) and use it to create another class representing a 3-D vector
2. Create a class Pets from a class Animals and further create a class Dog from Pets and add a method bark to class Dog
