## Section 01 - Sorting I

- Problems:
1. Selection sort
2. Bubble sort
3. Insertion sort

- Patterns learned:
1. Selection Sort: Find the minimum element and place it at its correct position
2. Bubble Sort: Repeatedly swap adjacent elements so the largest element moves to the end after each pass
3. Insertion Sort: Maintain a sorted left portion and insert the current element into its correct position
