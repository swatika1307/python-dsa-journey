## Section 07 - Basic Hashing

- Problems:
1. Basic hashing
2. Character hashing
3. Hashing using dictionary
4. Count frequency of each element
5. Highest and lowest frequency element

- Patterns learned:
1. Frequency Counting: Store occurrences using a dictionary/hash map.
2. Precomputation: Compute frequencies once and answer queries efficiently.
3. Fast Lookup: Hashing provides average O(1) search, insert and update.
4. Key-Value Mapping: Store data as (key → value) pairs.
5. Trade-off: Extra space is used to reduce time complexity.
