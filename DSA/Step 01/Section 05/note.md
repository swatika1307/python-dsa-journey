## Section 05 - Basic Maths

- Problems:
1. Count digits in a number
2. Reverse a number
3. Palindrome number
4. Armstrong number
5. Print all divisors
6. Check prime
7. GCD/HCF

- Patterns Learned:
1. Digit Extraction: digit = n % 10, n = n // 10
2. Build Number: rev = rev * 10 + digit
3. Square Root Optimization: Iterate till √n
4. Euclidean Algorithm: gcd(a, b) = gcd(b, a % b)
5. Time Complexity Awareness: O(n) → O(√n)
